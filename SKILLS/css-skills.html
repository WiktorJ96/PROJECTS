Podstawy CSS:

składnia CSS, selektory CSS, sposoby dodawania stylów do HTML, kaskadowość i specyficzność, dziedziczenie w CSS

Stylizacja tekstu i fontów:

właściwości fontów (font-size, color, font-style, font-weight, etc.), stylizacja tekstu (text-indent, text-align, text-transform, line-height), korzystanie z Google Fonts

Model pudełkowy i układ strony:

box model (margin, padding, border), box-sizing, elementy blokowe i liniowe, właściwość display, pozycjonowanie elementów (float, flexbox, grid)

Kolory i jednostki:

systemy kolorów (RGB, HEX, HSL), jednostki w CSS (px, em, rem, %, vw/vh)

Zaawansowane CSS:

pseudoklasy i pseudoelementy, zaawansowane selektory CSS, responsywne projektowanie

Techniki stylizacji i animacji:

wykorzystanie zmiennych CSS z użyciem selektora

:root

, tworzenie złożonych animacji CSS z wykorzystaniem

@keyframes

Zasoby i narzędzia:

korzystanie z Google Fonts, źródła darmowych grafik i wideo, narzędzia do tworzenia schematów kolorów, podstawy pracy z programami graficznymi

Praktyczne umiejętności:

tworzenie layoutów stron, stylizacja przycisków i menu, praca z projektami graficznymi (mockup), debugowanie kodu CSS, optymalizacja kodu CSS
