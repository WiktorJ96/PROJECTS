Podstawy SQL

SELECT, FROM, WHERE, podstawowa składnia, zasady korzystania z bazy danych.

Agregacje i Grupowanie

GROUP BY, HAVING, funkcje agregujące (MIN, MAX, SUM, COUNT, AVG), filtry.

Łączenia (JOINy)

INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, CROSS JOIN; kiedy i jak stosować.

Podzapytania (Subqueries)

Wbudowane zapytania w klauzulach SELECT, FROM, WHERE, użycie EXISTS i IN.

Klauzule Zaawansowane

WITH (CTE), klauzule rekurencyjne, PARTITION BY, OVER, funkcje okienkowe (window functions).

Transakcje

ACID, START TRANSACTION, COMMIT, ROLLBACK, poziomy izolacji, zarządzanie blokadami.

Indeksy i Optymalizacja

Tworzenie indeksów, EXPLAIN (plan zapytania), sposoby przyspieszania zapytań, statystyki.

Normalizacja

Formy normalne (1NF, 2NF, 3NF, BCNF), zasady projektowania struktur tabel dla minimalizacji duplikacji danych.

Funkcje SQL (Date, String, itd.)

Funkcje daty (DATE, TIME, TIMESTAMP), manipulacja ciągami znaków (CONCAT, SUBSTRING), konwersje typów.

Widoki (Views)

Tworzenie widoków, złożone zapytania jako widok, widoki z możliwością aktualizacji (updatable views).

Procedury i Funkcje Użytkownika

CREATE PROCEDURE, CREATE FUNCTION, parametry, pętle, warunki, zarządzanie logiką po stronie bazy.

Bezpieczeństwo i Uprawnienia

CREATE USER, GRANT, REVOKE, kontrola dostępu, szyfrowanie danych, audyt.

Replikacja i HA

Replikacja master-slave, failover, sharding, klastrowanie (np. Galera Cluster).

Zaawansowane Typy Danych

JSON, ARRAY, ENUM, typy przestrzenne (GIS), obsługa i indeksowanie danych specyficznych.

Performance Monitoring i Tuning

Narzędzia monitorujące (np. Performance Schema, pg_stat_statements), parametry konfiguracyjne, analiza logów.
