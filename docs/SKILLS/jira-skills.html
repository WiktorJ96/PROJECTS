Podstawy Jiry

Logowanie, interfejs, nawigacja, zakładanie konta w chmurze lub na serwerze.

Projekty i Tablice

Tworzenie projektów (Scrum, Kanban), konfiguracja tablic, kolumny, statusy zadań.

Tworzenie i Edycja Zadań

Rodzaje zadań (epic, story, task, bug), pola, priorytety, załączniki, komentowanie.

Workflow i Statusy

Definiowanie statusów, przejścia między statusami, konfiguracja warunków i reguł.

Backlog i Sprinty

Planowanie sprintów, priorytetyzacja backlogu, historie użytkownika (user stories).

Filtry i Zapytania (JQL)

Tworzenie zaawansowanych filtrów, Jira Query Language, zapisywanie i współdzielenie filtrów.

Raporty

Burndown Chart, Velocity Chart, raporty z postępu sprintu, status epików i historii.

Użytkownicy i Uprawnienia

Dodawanie nowych użytkowników, role, grupy, przydzielanie i ograniczanie uprawnień.

Konfiguracja Projektu

Schematy ekranów, typy zadań, pola niestandardowe, przepływ pracy (workflow).

Aplikacje i Integracje

Atlassian Marketplace, integracja z Confluence, Bitbucket, Slack, automatyzacje.

Administrowanie Systemem

Konfiguracja globalna, zarządzanie licencją, tworzenie schematów (workflow, ekranów, typów zadań) na poziomie całego systemu.

Automatyzacja Zadań

Reguły automatyzacji wbudowane w Jira, tworzenie zdarzeń (triggers), warunków i akcji.

Migracje i Import Danych

Import z innych narzędzi (Trello, Asana), migracje z serwera do chmury, eksport danych.

Skrypty i Rozszerzenia

ScriptRunner, tworzenie niestandardowych funkcji, integracja z REST API Jiry.

Zarządzanie Wydajnością

Monitorowanie obciążenia, optymalizacja przepływów pracy, dostosowanie planów hostingowych.
